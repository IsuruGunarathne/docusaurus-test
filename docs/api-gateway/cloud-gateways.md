# Cloud Gateways

Cloud gateways are fully managed by WSO2 and deployed in Bijira's cloud infrastructure. No gateway infrastructure management required.

## Benefits

- Zero-ops: WSO2 handles provisioning, scaling, and upgrades
- Global regions: Deploy closer to your users
- Built-in high availability and disaster recovery

## Selecting a Region

When creating an API proxy, select a cloud gateway region from the **Deploy** tab:

- `us-east-1` — US East (Virginia)
- `eu-west-1` — EU West (Ireland)
- `ap-southeast-1` — Asia Pacific (Singapore)
