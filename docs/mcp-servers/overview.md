# MCP Servers Overview

The Model Context Protocol (MCP) is an open standard that enables AI models to securely interact with external data sources and tools.

Bijira supports exposing your API proxies as MCP servers, making them accessible to AI agents like Claude, GPT-4, and others.

## Benefits

- Standardized interface for AI-to-API communication
- Built-in security and rate limiting
- No custom integration code required
