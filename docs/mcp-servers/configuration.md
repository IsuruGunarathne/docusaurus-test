# MCP Server Configuration

Configure your API proxy to act as an MCP server.

## Prerequisites

- An existing API proxy deployed in Bijira
- MCP-compatible AI client

## Configuration Steps

1. Navigate to your API proxy settings
2. Enable the **MCP Server** option
3. Configure allowed AI clients and scopes
4. Copy the MCP server endpoint URL
5. Provide the URL to your AI client configuration
