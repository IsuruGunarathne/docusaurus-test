# Policies

Policies are reusable configurations that control the behavior of your API proxy at request, response, or fault flows.

## Policy Types

- **Traffic Management** — Rate limiting, throttling, spike arrest
- **Security** — OAuth 2.0, JWT validation, API key enforcement
- **Mediation** — Request/response transformation, header manipulation
- **Observability** — Logging, tracing, analytics
