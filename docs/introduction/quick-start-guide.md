# Quick Start Guide

Get up and running with WSO2 API Platform in minutes.

## Prerequisites

- A WSO2 API Platform account ([sign up at console.bijira.dev](https://console.bijira.dev))
- Basic familiarity with REST APIs

## Step 1: Create an API Proxy

1. Log in to the WSO2 API Platform console
2. Click **Create API Proxy**
3. Select **From OpenAPI Specification**
4. Upload your OpenAPI file or enter the URL

## Step 2: Deploy the API

1. Click **Deploy** in the API Proxy editor
2. Select your target environment
3. Confirm deployment

## Step 3: Test the API

Use the built-in test console to send requests and verify your API proxy is working correctly.
