# What is WSO2 API Platform?

WSO2 API Platform (Bijira) is a next-generation, cloud-native API platform built for modern enterprises.

## Key Capabilities

- **API Proxy Development** — Create and manage REST, GraphQL, and WebSocket API proxies
- **Security** — OAuth 2.0, JWT, and API key-based authentication
- **Developer Portal** — Self-service API discovery and subscription management
- **AI Gateway** — Managed proxies for LLM providers
- **Self-Hosted Gateways** — Deploy gateways anywhere with centralized control
