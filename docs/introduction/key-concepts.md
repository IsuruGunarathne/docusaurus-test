# Key Concepts

## API Proxy
An API proxy acts as an intermediary between API consumers and backend services, providing security, transformation, and analytics.

## Organization
The top-level container in Bijira that groups all your API assets, environments, and team members.

## Environment
A deployment target (e.g., Development, Staging, Production) where your API proxies are deployed.

## Gateway
The runtime component that enforces policies and routes API traffic.
