# Test Using the Built-in Console

The Bijira console provides an integrated API testing interface powered by the Try-it feature.

## Steps

1. Open your API proxy in the Bijira console
2. Click the **Test** tab
3. Select an environment and endpoint
4. Provide authentication credentials
5. Fill in request parameters and body
6. Click **Execute** to see the response
