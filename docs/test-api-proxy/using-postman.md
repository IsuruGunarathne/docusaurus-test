# Test Using Postman

Export your API definition and test it using Postman.

## Steps

1. Export your API as an OpenAPI specification from the Bijira console
2. Import the spec into Postman using **Import** → **File**
3. Configure an environment with your gateway URL and credentials
4. Run requests against your deployed proxy
