# LLM Providers

The AI Gateway supports multiple LLM providers out of the box.

## Supported Providers

| Provider | Models |
|----------|--------|
| OpenAI | GPT-4o, GPT-4, GPT-3.5 |
| Anthropic | Claude 3.5 Sonnet, Claude 3 Opus |
| Azure OpenAI | GPT-4, GPT-3.5 |
| Google Gemini | Gemini 1.5 Pro, Gemini Flash |
| AWS Bedrock | Titan, Claude via Bedrock |

## Connecting a Provider

1. Navigate to **AI Gateway** → **Providers**
2. Click **Add Provider**
3. Select the provider type
4. Enter your API credentials
5. Configure rate limits and quotas
