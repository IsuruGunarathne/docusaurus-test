# GraphQL APIs

Create GraphQL API proxies to expose your GraphQL backends with governance and security.

## Create a GraphQL API Proxy

1. Navigate to **APIs** in the console
2. Click **Create API Proxy** → **My APIs** → **GraphQL API**
3. Upload your GraphQL schema (`.graphql` or `.sdl`)
4. Configure the backend GraphQL endpoint
5. Set field-level authorization if needed
6. Deploy to your environment

## Features

- Schema validation and introspection control
- Query depth limiting
- Field-level rate limiting
- Subscription support
