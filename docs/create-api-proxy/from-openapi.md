# Create an API Proxy from an OpenAPI Specification

Import an existing OpenAPI 3.x or Swagger 2.0 specification to automatically generate an API proxy.

## Steps

1. Navigate to **APIs** in the Bijira console
2. Click **Create API** → **Import OpenAPI**
3. Upload your `.yaml` or `.json` specification file
4. Review the generated endpoints and configure backend URL
5. Click **Create**
