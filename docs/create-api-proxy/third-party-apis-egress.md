# Third-Party APIs (Egress)

Create proxies for external APIs to add governance, security, and observability.

## Create an Egress API Proxy

1. Navigate to **APIs** in the console
2. Click **Create API Proxy** > **Third-Party APIs**
3. Enter the third-party API's base URL or OpenAPI spec
4. Configure authentication credentials
5. Apply policies as needed
6. Deploy the egress proxy
