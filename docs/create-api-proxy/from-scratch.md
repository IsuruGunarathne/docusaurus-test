# Create an API Proxy from Scratch

Build an API proxy manually by defining resources and methods without an existing specification.

## Steps

1. Navigate to **APIs** in the Bijira console
2. Click **Create API** → **Design New**
3. Define API name, version, and base path
4. Add resources and HTTP methods
5. Configure backend endpoints per resource
6. Click **Create**
