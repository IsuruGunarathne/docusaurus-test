# REST APIs

Create and manage REST API proxies on WSO2 API Platform.

## Create a REST API Proxy

1. Navigate to **APIs** in the console
2. Click **Create API Proxy** → **My APIs** → **REST API**
3. Provide your OpenAPI specification (file upload or URL)
4. Configure the backend endpoint
5. Set authentication and rate limiting policies
6. Deploy to your environment

## Supported Formats

- OpenAPI 3.x (YAML/JSON)
- Swagger 2.0 (YAML/JSON)
- Manual configuration (no spec required)
