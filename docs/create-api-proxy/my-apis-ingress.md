# My APIs (Ingress)

Expose your backend services through WSO2 API Platform as managed, governed API proxies.

## Create an Ingress API Proxy

1. Navigate to **APIs** in the console
2. Click **Create API Proxy** > **My APIs**
3. Choose your API type: REST, GraphQL, or WebSocket
4. Configure the backend URL
5. Set up authentication and rate limiting
6. Deploy to your environment
