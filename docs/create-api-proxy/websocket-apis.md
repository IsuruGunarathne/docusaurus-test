# WebSocket APIs

Create WebSocket API proxies to expose real-time, bi-directional services.

## Create a WebSocket API Proxy

1. Navigate to **APIs** in the console
2. Click **Create API Proxy** → **My APIs** → **WebSocket API**
3. Enter the WebSocket backend URL (`ws://` or `wss://`)
4. Configure topics and message flows
5. Set authentication policies
6. Deploy to your environment

## Use Cases

- Live notifications and event streams
- Chat and messaging applications
- Real-time data feeds (financial, IoT)
