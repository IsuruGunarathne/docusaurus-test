# Create API Proxy — Overview

WSO2 API Platform allows you to create API proxies for both your own APIs and third-party APIs.

## Types of API Proxies

- **My APIs (Ingress)** — Expose your backend services as managed APIs
- **Third-Party APIs (Egress)** — Create proxies for external APIs to add governance and security
