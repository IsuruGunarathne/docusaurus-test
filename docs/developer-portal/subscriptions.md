# Subscriptions

Developers subscribe to APIs to obtain credentials (API keys or OAuth clients) and gain access within defined usage limits.

## Subscribing to an API

1. Browse to the API in the Developer Portal
2. Click **Subscribe**
3. Select an application (or create a new one)
4. Choose a subscription plan
5. Click **Subscribe** — credentials are generated automatically

## Subscription Plans

Plans define rate limits and quotas:

| Plan | Requests/min | Monthly Quota |
|------|-------------|---------------|
| Free | 10 | 1,000 |
| Basic | 100 | 50,000 |
| Pro | 1,000 | Unlimited |
