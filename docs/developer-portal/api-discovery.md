# API Discovery

The Developer Portal provides a searchable catalog of all published APIs in your organization.

## Browsing APIs

- Search by name, tag, or category
- Filter by status (published, deprecated)
- View API documentation, try-it console, and usage examples

## Publishing an API

APIs are published to the Developer Portal from the Bijira console:

1. Open your API proxy
2. Navigate to **Lifecycle**
3. Click **Publish** to make it visible in the portal
